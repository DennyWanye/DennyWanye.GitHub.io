番茄工作法

· 2022-03-09 · # daily

一、番茄工作法的工具：

  1. 一个计时工具

  2. 一张today's TODO List，包含以下内容：

    2.1 小标题，包括时间、地点、人物

    2.2 清单，包含今天要做的事情，先后顺序排列

    2.3 计划外的紧急事件

  3. activity inventory：

    3.1 标题： 人物、事件

    3.2 想要做什么的事情

  4. 一份记录表，用于记录原始数据。包含：数据、摘要、完成任务所需要的番茄钟时间。

二、法则

规则一： 一个番茄时间共30分钟 = 25分钟工作时间 + 5分钟休息时间

规则二： 一个番茄时间不可分割

规则三：每四个番茄时间后，停止你的工作，进行一次较长时间的休息，15~30分钟

规则四：完成一个任务，划掉一个任务

三、番茄工作法流程

注意：

  1. 弄清楚完成某项任务所需要的时间
  2. 减少被打断的次数
  3. 估测某项任务所需要的时间
